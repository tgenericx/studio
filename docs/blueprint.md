# **App Name**: DayMode

## Core Features:

- Day Mode Selection: Allow users to select a day mode (Deep Work, Execution, Balanced, Chill) to define scheduling rules.
- Task Input: Enable users to add tasks with titles, durations (using a chip selector), and priority (must-do/optional).
- Event Scheduling: Allow users to add fixed events with start and end times, including prep and transition buffers.
- Automated Schedule Generation: Automatically generate a schedule based on the selected day mode, tasks, events, and scheduling algorithm.
- Timeline View: Display the generated schedule in a clear, mobile-optimized timeline view with hour markers and time blocks.
- Task Completion: Allow users to mark tasks as complete directly from the timeline view.

## Style Guidelines:

- Primary color: Deep blue (#2563EB) for a focused and structured feel.
- Background color: Very light blue (#E0F2FF), providing a clean and calm backdrop.
- Accent color: Teal (#2DD4BF) to highlight key interactive elements, and provide clear visual cues.
- Headline font: 'Space Grotesk' sans-serif for a techy and modern feel.
- Body font: 'Inter' sans-serif for readability and clean UI.
- Use 'lucide-react' icons to visually represent Day Modes.
- Mobile-first layout with full-width cards and components, optimized for one-handed use.
- Smooth transitions (200ms) between day modes.
